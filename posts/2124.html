网易云音乐解锁灰色歌曲
Publish: 2023/5/21   
  • 教程

什么是UnblockNeteaseMusic

UnblockCloudMusic是由nondanee编写的开源免费工具，其原理是通过使用 QQ / 虾米 / 百度 / 酷狗 / 酷我 / 咪咕 / JOOX 音源替换变灰歌曲链接

如何解灰

下载解灰需要的文件
观看视频教程

  • 导入证书
  1. 打开下载的ca.cer文件
  2. 点击安装证书，存储位置选择本地计算机
  3. 选择 将所有的证书都放入下列存储，存储位置选择受信任的根证书颁发机构
  4. 完成安装
  • 开机自启
  1. 将UnblockNeteaseMusic.exe找一个文件夹复制，并复制文件地址
  2. 将vbs脚本拖入开始文件夹
    （C:\ProgramData\Microsoft\Windows\Start Menu\Programs\Startup）
  3. 编辑脚本，将文件地址四个字替换成刚刚复制的文件地址，并删除多余的双引号
DIM objShell
set objShell = wscript.createObject("wscript.shell")
iReturn = objShell.Run("cmd.exe /C 文件地址", 0, TRUE)

“C:***\a.vbs” 删除双引号
4. 双击启动vbs脚本
（之后开机会自启）

  • 设置代理
  1. 打开网易云音乐
  2. 选择设置-工具，找到Http代理
  3. 选择自定义代理，服务器填localhost，端口填8080
  4. 重启网易云音乐即可解灰
